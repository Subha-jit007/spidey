# 🕷️ SUBHAJIT MAHATA — Friendly Neighborhood AI Developer

Spider-Verse comic-maximalist portfolio. No frameworks, no build step, no vibe-coded UI.

## Run locally
```
python -m http.server 8000
```
Open http://localhost:8000

## Deploy free
- **Vercel**: `npx vercel` in this folder, or drag-drop at vercel.com
- **GitHub Pages**: push to a repo → Settings → Pages → deploy from branch

## What's inside
- Verlet-physics spider swinging on a rope in the hero (canvas, hand-rolled — follows your mouse, click to kick it)
- Comic art masked through the hero typography
- Pinned scroll-storytelling origin story with iris-wipe panel transitions
- Char-by-char scrub reveal, pinned horizontal skills with velocity skew
- Parallax scrapbook collage, scroll-reactive marquees, web-splat cursor
- Spider riding a thread as the scroll progress bar
- Press `I` → Iron-Spider theme (Stark upgrade 🤖)

Stack: GSAP + ScrollTrigger + Lenis (CDN) · vanilla HTML/CSS/JS
